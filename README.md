# RAG-Based PDF Chatbot

Ask questions from any PDF using AI. Upload a document, ask a question in plain English, and get a context-aware answer generated by Groq LLM — answers grounded in your actual document, not hallucinated.

---

## Features

- Upload single or multiple PDF documents
- Semantic similarity search over document content
- RAG pipeline: retrieval + generation, not just generation
- Fast inference via Groq LLM (near real-time responses)
- FAISS vector store for efficient embedding search
- Clean, interactive Streamlit UI
- Modular codebase — helper, vectorstore, and LLM layers separated

## How It Works

```
PDF Upload → Text Extraction → Chunking → Embedding
                                              |
User Query → Semantic Search → Relevant Chunks → Groq LLM → Answer
```

1. PDF text is extracted and split into overlapping chunks
2. Each chunk is converted to a vector embedding
3. Embeddings are stored in a FAISS index
4. On a user query, the top-k most relevant chunks are retrieved
5. Retrieved context and query are sent to Groq LLM for answer generation

## Run Locally

**Clone the repository**

```bash
git clone https://github.com/aps-codes/ai-pdf-chatbot-rag.git
cd ai-pdf-chatbot-rag
```

**Install dependencies**

```bash
pip install -r requirements.txt
```

**Set up environment variables**

Create a `.env` file in the project root:

```env
GROQ_API_KEY=your_groq_api_key_here
```

Generate a free API key at: https://console.groq.com

**Run the app**

```bash
streamlit run app.py
```

The app will open at `http://localhost:8501`
